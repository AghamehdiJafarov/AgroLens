# AgroLens — оценка качества плодоовощной продукции по фото

Первый из двух сайтов проекта. По фотографии плода или партии считает состав
цвета, находит тёмные зоны (вероятные дефекты) и выставляет сорт A / B / C по
стандарту, который задаёт сам пользователь под своего покупателя.

Интерфейс на трёх языках: русский, английский, азербайджанский.
Палитра природная: листовой зелёный, спелый янтарь, земля, кремовый фон.

## Что делает честно

Всё, что показывает инструмент, — результат реальных вычислений над пикселями:

- **Состав цвета** — доля красного, оранжевого, жёлтого, зелёного и тёмного/бурого
  на плоде. Считается по среднему оттенку каждого пикселя (HSV). Точная арифметика.
- **Тёмные зоны** — доля площади плода заметно темнее основного тона. Сигнал пятен,
  гнили, побитости. Порог чувствительности настраивается.
- **Калибр** — насколько плод занимает кадр. Это **относительная** величина;
  абсолютные миллиметры требуют эталона в кадре (например, монеты).
- **Сорт A/B/C** — строго по порогам, которые ввёл пользователь. Прозрачно и
  воспроизводимо: одинаковое фото + одинаковые пороги = одинаковый сорт.

## Чего НЕ делает (и не притворяется)

Не отличает болезни друг от друга, не даёт миллиметры без эталона, не заявляет 99%
точности, не заменяет лабораторию. Эти границы честно перечислены прямо на сайте.

## Приватность

Фотография не покидает устройство. Весь анализ — в браузере на canvas. Нет сервера,
нет загрузки фото, нет API-ключа.

## Видео и презентация

Как добавить свой промо-ролик и PDF-презентацию — подробная инструкция лежит в
папке `public/README-положи-сюда-видео-и-презентацию.txt`. Коротко:

- **Видео**: загрузи ролик на YouTube, вставь его ID в переменную `YT_ID` в начале
  `src/Landing.jsx`. Пока ID не вставлен, показывается аккуратная заглушка.
- **Презентация**: положи PDF с именем `AgroLens.pdf` в папку `public/` — кнопки
  "Скачать презентацию" начнут его отдавать.

## Как запустить

Нужен Node.js 18+.

```bash
npm install     # поставить зависимости
npm run dev     # локально → http://localhost:5173
npm run build   # собрать в папку dist/ для хостинга
npm run preview # посмотреть собранное
```

## Структура

```
src/
  main.jsx      — точка входа, маршруты / и /app
  lang.jsx      — переключатель RU/EN/AZ + весь словарь переводов
  Landing.jsx   — главная (в начале файла: YT_ID и DECK_FILE для видео и PDF)
  App.jsx       — инструмент: стандарт, загрузка фото, результат
  analyze.js    — движок анализа изображения (вся честная математика здесь)
  index.css     — шрифты и базовый фон
public/         — сюда кладётся AgroLens.pdf; попадает в корень сайта
```

## Технологии

React 18 · Vite · Tailwind CSS · Recharts · lucide-react. Тот же стек, что и во
втором сайте проекта, — чтобы оба выглядели как одна система.
