Как добавить своё видео и презентацию
======================================

ВИДЕО (промо-ролик на главной странице)
----------------------------------------
Видео берётся с YouTube — так оно не утяжеляет сайт и работает на всех устройствах.

1. Загрузи свой ролик на YouTube (можно поставить видимость "Доступ по ссылке",
   если не хочешь, чтобы его находили в поиске).
2. Открой ролик, скопируй его ID — это часть ссылки после watch?v=
   Пример: в ссылке  https://www.youtube.com/watch?v=dQw4w9WgXcQ
   ID = dQw4w9WgXcQ
3. Открой файл  src/Landing.jsx  в любом текстовом редакторе.
4. В самом начале файла найди строку:
        const YT_ID = "REPLACE_WITH_YOUTUBE_ID";
5. Замени REPLACE_WITH_YOUTUBE_ID на свой ID (кавычки оставь):
        const YT_ID = "dQw4w9WgXcQ";
6. Сохрани файл. Готово — на месте заглушки появится твой ролик.

Пока ID не заменён, на сайте вместо видео показывается аккуратная заглушка
с подсказкой — это нормально, сайт из-за этого не ломается.


ПРЕЗЕНТАЦИЯ (PDF, который скачивается по кнопке)
-------------------------------------------------
1. Сохрани свою презентацию в формате PDF.
2. Назови файл  AgroLens.pdf
3. Положи его в эту самую папку (public/), рядом с этим текстовым файлом.
4. Готово — кнопки "Скачать презентацию" на сайте начнут отдавать этот файл.

Если хочешь другое имя файла — открой src/Landing.jsx, найди в начале строку:
        const DECK_FILE = "/AgroLens.pdf";
и поменяй имя (слэш в начале оставь).

Всё, что лежит в папке public/, попадает в корень сайта как есть.
